# Example Input

The example input video is a short sign language clip from [SignSuisse](https://www.signsuisse.sgb-fss.ch/).

**URL:**
```
https://www.sgb-fss.ch/signsuisse/fileadmin/signsuisse_ressources/videos/262C81F5-FB9D-759D-08E1CB201ADEB239.mp4
```

To download it to `data/input/test.mp4`, run:

```bash
bash scripts/download_test_video.sh
```
